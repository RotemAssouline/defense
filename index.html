Chapter 1: Lagrangians

A Lagrangian \(L\) on a manifold \(M\) is a smooth function on the tangent bundle \(TM\). The action of a curve \(\gamma:[0,T]\to M \)   is: \[\mathrm{A}(\gamma) : = \int_0^T L(\dot\gamma(t))dt.\]

An extremal of \(L\) is a curve which minimizes the action \(\mathrm{A}\) among all curves with the same endpoints.

🥴 A Lagrangian assigns a value to each position and velocity. An extremal is a path for which the sum of these values over the entire journey is the smallest possible.

examples

  • Lagrangians of the form \(L = T - V\), where \(T\) is kinetic energy (i.e. quadratic in the velocity) and \(V\) is potential energy.
    • Riemannian metrics - just kinetic energy;
      action ~ length\(^2\), extremals - geodesics.
    • Mechanical Lagrangians - \(V\) is a potential depending only on position.
    • Electromagnetic Lagrangians - \(V\) is an affine function of the velocity.
  • Finsler metrics: 2-homogeneous in the velocity
  • \(\phi(F)\), where \(F\) is a Finsler metric.
\[M : = \left\{x = (x_1,\dots,x_k) \in (\mathbb{R}^d)^k \,\, \mid \,\, x_i \ne x_j \, \, \forall \, i\ne j\right\}. \] \[L = \frac12\sum_{i=1}^km_i\sum_{\ell=1}^d(dx_i^\ell)^2 + \frac12 + \underbrace{\sum_{i=1}^k\sum_{j=i+1}^k\frac{Gm_im_j}{|x_i-x_j|}}_{U},\]
\(G\) – gravitational constant, \(m_i\) – masses, \(x_i = (x_i^1,\dots,x_i^d)\) – position of the \(i\)-th body.

\(\,\,\,\mathrm{Ric}_{\mathrm{Vol},\infty} = -\Delta U + U^{-1}|\nabla U|^2 - (2U)^{-2}(dU)^2 \ge 0.\)

Chapter 2: Ricci curvature

A Riemannian manifold is a smooth manifold equipped with a Riemannian metric , which is a smooth choice of inner product on each tangent space.

🥴 A Riemannian manifold is a space which looks like Euclidean space, but has a different way of measuring distances and angles.

A geodesic on a Riemannian manifold is (at least locally) the shortest path between a pair of points on the manifold.

🥴 Geodesics are like straight lines, but on a curved space.

The Ricci curvature of a Riemannian metric is a trace of the Riemann Curvature tensor.
Ricci curvature controls the way volume is distorted along families of geodesics..
More precisely, if \(v\) is a tangent vector at a point \(p \in M\), and \(V \) is a vector field such that \[\nabla_VV = 0, \qquad V\vert_p = v \qquad \text{ and } \qquad \nabla V\vert_p = 0,\] then \(V\mathrm{div} V = -\mathrm{Ric}(V) .\)

🥴 If we take a little object and let it flow along parallel geodesics in the direction \(v\), then the second logarithmic derivative of the object's volume will be \(-\mathrm{Ric}(v)\) .

Example   1:   The Euclidean plane   (  \(\mathrm{Ric} = 0\)   )

Example   2:   The hyperbolic plane   (  \(\mathrm{Ric} = -1\)   )

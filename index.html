Background

Part 1: Lagrangians

A Lagrangian \(L\) on a manifold \(M\) is a smooth function on the tangent bundle \(TM\). The action of a curve \(\gamma:[0,T]\to M \)   is: \[\mathrm{A}(\gamma) : = \int_0^T L(\dot\gamma(t))dt.\]

An extremal of \(L\) is a curve which minimizes the action \(\mathrm{A}\) among all curves with the same endpoints.

🥴 A Lagrangian assigns a value to each position and velocity. An extremal is a path for which the integral of these values over the entire journey is the smallest possible.

examples

  • \(L = T - V\), where \(T\) is kinetic energy (i.e. quadratic in the velocity) and \(V\) is potential energy.
    • Riemannian metrics: just kinetic energy;
      action ~ length\(^2\), extremals - geodesics.
    • Mechanical Lagrangians: \(V\) is a potential depending only on position.
  • Electromagnetic Lagrangians: \(L = T + qA\cdot \mathrm{v}\)  .
  • Finsler metrics: 2-homogeneous in the velocity.
  • Isotropic:   \(\phi(F)\), where \(F\) is Riemannian/Finslerian.
The Hamiltonian associated to a Lagrangian \(L\) is the function \(H\) on \(T^*M\) defined by \[H(p) : = \sup_{v \in T_xM}p(v) - L(v), \quad p\in T^*_xM,\,\,x \in M.\] The tangent vector where the above supremum is achieved is denoted by \(\mathcal{L}p\), and we call the map \(\mathcal{L} : T^*M \to T_xM\) the Legendre transform associated to \(L\).
The energy is the function on \(TM\) defined by \[E(v) : = H(\mathcal{L}^{-1} v), \qquad v \in TM.\]
\[M : = \left\{x = (x_1,\dots,x_k) \in (\mathbb{R}^d)^k \,\, \mid \,\, x_i \ne x_j \, \, \forall \, i\ne j\right\}. \] \[L = \frac12\sum_{i=1}^km_i\sum_{\ell=1}^d(dx_i^\ell)^2 + \frac12 + \underbrace{\sum_{i=1}^k\sum_{j=i+1}^k\frac{Gm_im_j}{|x_i-x_j|}}_{U},\]
\(G\) – gravitational constant, \(m_i\) – masses, \(x_i = (x_i^1,\dots,x_i^d)\) – position of the \(i\)-th body.

\(\,\,\,\mathrm{Ric}_{\mathrm{Vol},\infty} = -\Delta U + U^{-1}|\nabla U|^2 - (2U)^{-2}(dU)^2 \ge 0.\)
\[L(x,v) = \frac{|v|^2}{2} + \frac12 + A(x,v),\]
\(A\) - Magnetic field - Linear in \(v\).

More generally: \[ L = g/2 + 1/2 + \eta\] where \(g\) is a Riemannian metric and \(\eta\) is a one-form.

Part 2: Optimal Transport

\(\mu_0,\mu_1\) - probability measures on some measure space \(\mathcal{X}\).
\(\mathrm{c}: \mathcal{X} \times \mathcal{X} \to \mathbb{R}\)  - some cost function.
The optimal transport (Monge-Kantorovich) problem is the problem of finding a map \(T:\mathcal{X}\to\mathcal{X}\)   minimizing \[ \int \mathrm{c}(x_0,T(x_0)) d\mu_0(x_0)\] among all maps pushing forward \(\mu_0\) to \(\mu_1\).
Such \(T\) is called an optimal transport map .

🥴 Optimal transport seeks the most cost-effecient way to redistribute mass, goods, etc. from one configuration to another.

Given a Lagrangian \(L\) on a manifold \(M\), we can define a cost function by setting \[\mathrm{c}(x_0,x_1) : = \inf_\gamma \mathrm{A}(\gamma), \qquad x_0,x_1 \in M,\] where the infimum is over all curves \(\gamma\)   joining \(x_0\) to \(x_1\), and \(\mathrm{A}\) is the action.

If \(L\) is a Riemannian metric, then \(\mathrm{c}\) is distance.

Theorem: Under classical assumptions on the Lagrangian \(L\), for every pair \(\mu_0,\mu_1\) of absolutely-continuous, compactly supported probability measures on \(M\), there exists an optimal transport map \(T\) from \(\mu_0\) to \(\mu_1\).

In fact, there exists a family \(\{T_\lambda\}_{0 \le \lambda \le 1}\)   of maps such that

  • \(T_0\) is the identity map and \(T_1\) is an optimal transport map from \(\mu_0\) to \(\mu_1\);
  • For each \(0 \le \lambda\le 1\), the map \(T_\lambda\) is an optimal transport map from \(\mu_0\) to \(\mu_\lambda : = (T_\lambda)_*\mu_0\)  .
  • for each \(x \in M\), the curve \(\lambda\mapsto T_\lambda(x)\)   is an action-minimizing extremal of \(L\).
  • There exists a viscosity subsolution \(u\) to the Hamilton-Jacobi equation, and \(\ell : M \to [0,\infty)\)   such that \[T_\lambda = \pi\circ\Phi_{\lambda\ell}\circ du \qquad \text{\(\mu_0\) -a.e.}\]

🥴

We can solve the OT problem for costs arising from reasonable Lagrangians.

  • The solution describes not only where each piece of goods should end up, but also along what path it should be transported.
  • The resulting one-parameter family of maps (obtained by stopping the paths at intermediate times) are all optimal.
  • The above solution can be read off from a "guiding function", and the transport trajectories are "gradient curves" of this function (in a Lagrangian sense).

Part 3: Ricci curvature

A Riemannian manifold is a smooth manifold equipped with a Riemannian metric , which is a smooth choice of inner product on each tangent space.

🥴 A Riemannian manifold is a space which looks like Euclidean space, but has a different way of measuring distances and angles.

A geodesic on a Riemannian manifold is (at least locally) the shortest path between a pair of points on the manifold.

🥴 Geodesics are like straight lines, but on a curved space.

The Ricci curvature of a Riemannian metric is a trace of the Riemann Curvature tensor.
  • Ricci curvature controls the way volume is distorted along families of geodesics.
  • If \(v\) is a tangent vector at a point \(p \in M\), and \(V \) is a vector field such that \[\nabla_VV = 0, \qquad V\vert_p = v \qquad \text{ and } \qquad \nabla V\vert_p = 0,\] then \(V\mathrm{div} V = -\mathrm{Ric}(V) .\)

🥴 If we take a little object and let it flow along parallel geodesics in the direction \(v\), then the second logarithmic derivative of the object's volume will be \(-\mathrm{Ric}(v)\) .

Example   1:   The Euclidean plane   (  \(\mathrm{Ric} = 0\)   )

Example   2:   The hyperbolic plane   (  \(\mathrm{Ric} = -1\)   )
